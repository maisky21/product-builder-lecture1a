# 오늘 뭐 먹지? (Dinner Menu Recommender)

저녁 메뉴 선택 고민을 해결해주는 간단한 웹 사이트입니다.

## 주요 기능
- **랜덤 메뉴 추천**: 버튼을 클릭하면 다양한 메뉴 중 하나를 무작위로 추천합니다.
- **카테고리 필터**: 한식, 양식, 일식/중식, 간단식 중 원하는 종류를 선택하여 추천받을 수 있습니다.
- **반응형 디자인**: 모바일과 데스크탑 모두에서 최적화된 화면을 제공합니다.

## 기술 스택
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

## 배포 방법 (GitHub Pages)
1. GitHub 리포지토리를 생성합니다.
2. 로컬 저장소를 연결합니다: `git remote add origin <리포지토리-URL>`
3. 코드를 푸시합니다: `git push -u origin main`
4. 리포지토리의 `Settings > Pages` 메뉴에서 `main` 브랜치를 선택하여 배포합니다.
